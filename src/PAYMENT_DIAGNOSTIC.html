🔍 PayStack Payment Diagnostic Tool

Use this tool to diagnose payment verification issues.

Step 1: Check Edge Function Status

Step 2: Verify a Payment Reference

Step 3: Check Database (Payments Table)

Go to your Supabase Dashboard → Table Editor → payments table
Look for your payment reference and check the status column.

Step 4: Check Edge Function Logs

Run this command in your terminal:

cd supabase
supabase functions logs payments-simple --follow

Then try making a test payment and watch the logs in real-time.

Common Issues & Solutions

  • Edge Function not deployed: Run cd supabase && supabase functions deploy payments-simple
  • Wrong callback URL: Check that FRONTEND_URL environment variable matches your deployed site
  • PayStack keys not set: Check that PAYSTACK_SECRET_KEY is set in Supabase Edge Function secrets
  • Test card not working: Use 4084 0840 8408 4081 with any future expiry and any CVV
