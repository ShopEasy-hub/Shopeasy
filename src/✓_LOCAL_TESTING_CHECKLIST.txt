COMMON ISSUES & FIXES:
──────────────────────
❌ Blank page when clicking "Continue to Payment"
   → Check browser console for error
   → Redeploy Edge Function

❌ 404 error after payment callback
   → Make sure App.tsx is saved
   → Restart dev server (Ctrl+C then npm run dev)

❌ "Payment gateway not configured"
   → Set PAYSTACK_SECRET_KEY in Supabase secrets

❌ CORS error
   → Redeploy Edge Function
   → Check Edge Function has CORS headers

❌ "Unauthorized" error
   → Make sure you're logged in
   → Check access token is being sent

❌ Webhook not firing
   → This is OK for local testing
   → Webhook should still work (Edge Function is deployed)
   → Check Paystack Dashboard → Settings → Webhooks
